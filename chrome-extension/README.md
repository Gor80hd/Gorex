# Gorex Chrome Extension

Расширение для Google Chrome, которое позволяет скачивать видео с любого поддерживаемого сайта одной кнопкой через приложение **Gorex**.

## Возможности

- **Кнопка скачивания** на панели браузера — работает на YouTube, TikTok, Instagram, VK, Rutube и 30+ сервисах
- **Выбор качества** — 4K, 1080p, 720p и т.д. (форматы загружаются от yt-dlp)
- **Только аудио** — скачать MP3 без видео одним переключателем
- **Обрезка** — задать начало и конец фрагмента перед скачиванием
- **Скачать превью** — сохранить обложку/миниатюру видео
- **Очередь прямо в браузере** — всплывающий оверлей показывает статус загрузки
- **Кнопка «Запустить»** — если Gorex не запущен, отображается подсказка

## Структура

```
chrome-extension/
  manifest.json          — манифест расширения (MV3)
  background.js          — service worker (связь с Gorex API)
  popup/
    popup.html           — интерфейс попапа
    popup.js             — логика попапа
    popup.css            — стили попапа
  content/
    content.js           — контент-скрипт (очередь-оверлей на странице)
    content.css          — стили оверлея
  icons/
    icon16.png  …  icon128.png
```

## Установка (Developer Mode)

1. Откройте `chrome://extensions/`
2. Включите **«Режим разработчика»** (правый верхний угол)
3. Нажмите **«Загрузить распакованное»**
4. Выберите папку `chrome-extension/` из этого репозитория
5. Закрепите расширение на панели инструментов

## Как работает

Расширение общается с десктопным приложением Gorex через локальный HTTP API на `127.0.0.1:19870` (loopback-only, недоступен извне).

### API эндпоинты (порт 19870–19875)

| Метод | Путь | Описание |
|-------|------|----------|
| GET | `/gorex-api/ping` | Проверка статуса приложения |
| GET | `/gorex-api/formats?url=...` | Получить форматы видео через yt-dlp |
| POST | `/gorex-api/queue/add` | Добавить URL в очередь загрузки |
| GET | `/gorex-api/queue` | Получить текущую очередь |

### POST `/gorex-api/queue/add` — тело запроса

```json
{
  "url":               "https://...",
  "formatId":          "137",       // ID формата из /formats, null = наилучшее
  "audioOnly":         false,
  "clipStart":         30,          // секунды, null = начало
  "clipEnd":           90,          // секунды, null = конец
  "downloadThumbnail": false
}
```

## Требования

- Gorex v2.2.0+ (с поддержкой Extension API)
- Google Chrome 116+ (Manifest V3)

## Безопасность

- API сервер слушает **только на 127.0.0.1** (loopback)
- CORS разрешает запросы только от `chrome-extension://` и localhost
- Все запросы к API проходят через фоновый service worker — CSP popup не допускает прямых `fetch` к localhost
